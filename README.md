# Pizza Hut Sales Analysis (SQL)

## Overview
A practical SQL project that analyzes Pizza Hut sales data to uncover insights on customer behavior, revenue trends, and product performance using real-world transactional data.

---

## What This Project Covers
- End-to-end SQL analysis using relational data
- Business-focused insights from sales and order data
- Clear use of basic, intermediate, and advanced SQL concepts

---

## Dataset
Four interconnected tables:
- Orders
- Order_Details
- Pizzas
- Pizza_Types

---

## Key Analysis
- Total orders and total revenue
- Best-selling pizzas and most popular sizes
- Category-wise sales performance
- Peak ordering hours
- Revenue contribution by pizza type
- Cumulative revenue over time
- Top revenue-generating pizzas per category

---

## SQL Skills Demonstrated
- JOINs and aggregations
- GROUP BY, ORDER BY
- Subqueries
- Window functions (`RANK`, `SUM OVER`)
- Common Table Expressions (CTEs)

---

## Outcome
Delivered actionable insights that can support pricing decisions, inventory planning, and marketing strategies.
